Tipps:

- Die Fontawesome-Icons (https://fontawesome.com/) können folgendermaßen eingebunden werden:
  <link rel="stylesheet" href="css/all.min.css">

- Auch SVGs können als Favicon eingebunden werden:
  <link rel="shortcut icon" href="img/html5.svg" type="image/svg+xml">

- Die benötigten Farbwerte sind bereits im CSS (style.css) als Variablen hinterlegt.
